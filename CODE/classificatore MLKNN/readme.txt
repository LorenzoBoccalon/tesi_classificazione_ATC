questa cartella contiene i file riguardanti l'uso di MLKNN

	-classMLKNN.mat: contiene tutte le variabili di ambiente di classificatoreMLKNN.m
	-pureMLKNN.mat: contiene tutte le variabili di ambiente di pureMLKNN.m

	-paper.pdf: è l'articolo scientifico riguardante il modulo MLKNN

	-classificatoreMLKNN.m: è uno script dove si utilizza solo MLKNN senza LSTM
	-classificatoreLSTMLKNN.m: è uno script dove si utilizza MLKNN com LSTM

	-performance.txt: contiene i performance values di classificatoreLSTM.m
